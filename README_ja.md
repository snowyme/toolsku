# 機能概要

**toolsku** は **Next.js 15 + React 19** で構築されたモダンなオンラインツールボックスです。クライアントサイド（ブラウザローカル）処理を中心に設計されており、画像、PDF、動画、テキスト、JSON、エンコード/暗号化、Web/SEO、開発者ツール、ドキュメントなど数百の実用的なツールを網羅しています。データ処理は可能な限りローカルで完了し、アップロード不要でプライバシーを保護します。一部の高負荷処理（PDF→Word/Excel 変換など）は Alibaba Cloud DocMind プロキシ経由で実行されます。

### コア技術

- **Next.js 15（App Router）+ React 19 + TypeScript**、静的エクスポート（SSG）、任意の静的ホスティング / Alibaba Cloud Function Compute にデプロイ可能。
- **国際化**：`next-intl` により **簡体字中国語、繁体字中国語、English、日本語** の 4 言語に対応（`localePrefix: always`）。
- **ローカル演算**：WebAssembly（ffmpeg.wasm、PDF 解析）、Web Crypto API、Canvas、Web Audio などを活用し、ブラウザ上で変換・計算を実行。
- **検索インデックス自動生成**：ビルド時に多言語ツール検索インデックスと SEO ページを生成。

### 特徴

- **プライバシー最優先**：ほとんどのツールはブラウザ内で完全に動作し、ファイルをサーバーにアップロードしません。
- **多言語 SEO**：各ツールは 4 言語の静的ページと `hreflang` タグを生成し、検索エンジンに最適化。
- **バックエンド不要（デフォルト）**：静的サイトとしてそのまま利用可能。PDF→Word/Excel 変換のみ DocMind プロキシのデプロイが必要。
- **レスポンシブデザイン**：デスクトップとモバイルの両方に対応。
